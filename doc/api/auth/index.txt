Sign in user
------------

Parameters:
  * email - User email
  * password - User password

Request:
  POST /auth/sign_in
  Content-Type: application/json

  {"email":"test@example.com","password":"password"}

Response:
  Status: 200 OK
  Content-Type: application/json; charset=utf-8
  access-token: J9Y6KBOCd_yupH9XQSsTQA
  client: DZ2PwfFe5p671mv5eezpwQ
  uid: test@example.com

  {
    "data": {
      "id": 355,
      "email": "test@example.com",
      "company_id": 1216,
      "provider": "email",
      "uid": "test@example.com",
      "role": "admin"
    }
  }
