Create a company
----------------

Parameters:
  * type - Should always be set to <code>companies</code>
  * name - Company Name

Request:
  POST /companies
  Content-Type: application/vnd.api+json

  {"data":{"type":"companies","attributes":{"name":"ACME Company"}}}

Response:
  Status: 201 Created
  Content-Type: application/vnd.api+json

  {"data":{"id":"1755","type":"companies","links":{"self":"http://example.org/companies/1755"},"attributes":{"name":"ACME Company","node-counts":[{"node_type":"network-sites","count":0,"node_type_pretty":"Sites"},{"node_type":"buildings","count":0,"node_type_pretty":"Buildings"}]},"relationships":{"network-sites":{"links":{"self":"http://example.org/companies/1755/relationships/network-sites","related":"http://example.org/companies/1755/network-sites"}},"buildings":{"links":{"self":"http://example.org/companies/1755/relationships/buildings","related":"http://example.org/companies/1755/buildings"}},"users":{"links":{"self":"http://example.org/companies/1755/relationships/users","related":"http://example.org/companies/1755/users"}}}}}


Delete a company
----------------

Parameters:
  * type - Should always be set to <code>companies</code>
  * name - Company Name
  * id - The id of the company

Request:
  DELETE /companies/1756
  Content-Type: application/vnd.api+json

  {"data":{"type":"companies","id":"1756"}}

Response:
  Status: 204 No Content
