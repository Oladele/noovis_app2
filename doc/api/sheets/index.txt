Delete a sheet
--------------

Parameters:
  * workbook - Workbook id
  * building - Building id
  * type - Should always be set to <code>sheets</code>
  * name - Sheet Name
  * created-at - Sheet created at
  * updated-at - Sheet created at
  * record-count - Cable Run count for Sheet
  * id - The id of the sheet

Request:
  DELETE /sheets/31
  Content-Type: application/vnd.api+json

  {"data":{"relationships":{"workbook":{"data":{"type":"workbooks","id":39}},"building":{"data":{"type":"buildings","id":39}}},"type":"sheets","id":"31"}}

Response:
  Status: 204 No Content
