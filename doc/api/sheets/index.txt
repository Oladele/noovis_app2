Update a sheet
--------------

Parameters:
  * workbook - Workbook id
  * building - Building id
  * type - Should always be set to <code>sheets</code>
  * name - Sheet Name
  * created-at - Sheet created at
  * updated-at - Sheet created at
  * id - The id of the sheet

Request:
  PATCH /sheets/1296
  Content-Type: application/vnd.api+json

  {"data":{"relationships":{"workbook":{"data":{"type":"workbooks","id":1679}},"building":{"data":{"type":"buildings","id":1757}}},"type":"sheets","attributes":{"name":"Updated Sheet Name"},"id":"1296"}}
