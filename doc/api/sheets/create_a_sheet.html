Sheets API

Create a sheet

POST /sheets

Parameters

Name Description
relationships[workbook] Workbook id
relationships[building] Building id
type Should always be set to <code>sheets</code>
attributes[name] Sheet Name
attributes[created-at] Sheet created at
attributes[updated-at] Sheet created at
attributes[record-count] Cable Run count for Sheet
