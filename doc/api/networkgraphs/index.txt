Update a network-graph
----------------------

Parameters:
  * type - Should always be set to <code>network-graphs</code>
  * sheet - Sheet id
  * network-template - Network Template id
  * id - The id of the network-graph

Request:
  PATCH /network-graphs/120
  Content-Type: application/vnd.api+json

  {"data":{"type":"network-graphs","relationships":{"sheet":{"data":{"type":"sheets","id":263}},"network-template":{"data":{"type":"network-templates","id":184}}},"id":"120"}}

Response:
  Status: 200 OK
  Content-Type: application/vnd.api+json

  {"data":{"id":"120","type":"network-graphs","links":{"self":"http://example.org/network-graphs/120"},"attributes":{"nodes":[],"edges":[]},"relationships":{"sheet":{"links":{"self":"http://example.org/network-graphs/120/relationships/sheet","related":"http://example.org/network-graphs/120/sheet"}},"network-template":{"links":{"self":"http://example.org/network-graphs/120/relationships/network-template","related":"http://example.org/network-graphs/120/network-template"}},"nodes":{"links":{"self":"http://example.org/network-graphs/120/relationships/nodes","related":"http://example.org/network-graphs/120/nodes"}}}}}
