Update a network-graph
----------------------

Parameters:
  * type - Should always be set to <code>network-graphs</code>
  * sheet - Sheet id
  * network-template - Network Template id
  * id - The id of the network-graph

Request:
  PATCH /network-graphs/33
  Content-Type: application/vnd.api+json

  {"data":{"type":"network-graphs","relationships":{"sheet":{"data":{"type":"sheets","id":78}},"network-template":{"data":{"type":"network-templates","id":52}}},"id":"33"}}

Response:
  Status: 200 OK
  Content-Type: application/vnd.api+json

  {"data":{"id":"33","type":"network-graphs","links":{"self":"http://example.org/network-graphs/33"},"attributes":{"nodes":[],"edges":[]},"relationships":{"sheet":{"links":{"self":"http://example.org/network-graphs/33/relationships/sheet","related":"http://example.org/network-graphs/33/sheet"}},"network-template":{"links":{"self":"http://example.org/network-graphs/33/relationships/network-template","related":"http://example.org/network-graphs/33/network-template"}},"nodes":{"links":{"self":"http://example.org/network-graphs/33/relationships/nodes","related":"http://example.org/network-graphs/33/nodes"}}}}}


Create a network graph
----------------------

Parameters:
  * type - Should always be set to <code>network-graphs</code>
  * sheet - Sheet id
  * network-template - Network Template id

Request:
  POST /network-graphs
  Content-Type: application/vnd.api+json

  {"data":{"type":"network-graphs","relationships":{"sheet":{"data":{"type":"sheets","id":79}},"network-template":{"data":{"type":"network-templates","id":53}}}}}

Response:
  Status: 201 Created
  Content-Type: application/vnd.api+json

  {"data":{"id":"34","type":"network-graphs","links":{"self":"http://example.org/network-graphs/34"},"attributes":{"nodes":[],"edges":[]},"relationships":{"sheet":{"links":{"self":"http://example.org/network-graphs/34/relationships/sheet","related":"http://example.org/network-graphs/34/sheet"}},"network-template":{"links":{"self":"http://example.org/network-graphs/34/relationships/network-template","related":"http://example.org/network-graphs/34/network-template"}},"nodes":{"links":{"self":"http://example.org/network-graphs/34/relationships/nodes","related":"http://example.org/network-graphs/34/nodes"}}}}}
