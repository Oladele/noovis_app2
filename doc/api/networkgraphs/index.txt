Update a network-graph
----------------------

Parameters:
  * type - Should always be set to <code>network-graphs</code>
  * sheet - Sheet id
  * network-template - Network Template id
  * id - The id of the network-graph

Request:
  PATCH /network-graphs/68
  Content-Type: application/vnd.api+json

  {"data":{"type":"network-graphs","relationships":{"sheet":{"data":{"type":"sheets","id":166}},"network-template":{"data":{"type":"network-templates","id":108}}},"id":"68"}}

Response:
  Status: 200 OK
  Content-Type: application/vnd.api+json

  {"data":{"id":"68","type":"network-graphs","links":{"self":"http://example.org/network-graphs/68"},"attributes":{"nodes":[],"edges":[]},"relationships":{"sheet":{"links":{"self":"http://example.org/network-graphs/68/relationships/sheet","related":"http://example.org/network-graphs/68/sheet"}},"network-template":{"links":{"self":"http://example.org/network-graphs/68/relationships/network-template","related":"http://example.org/network-graphs/68/network-template"}},"nodes":{"links":{"self":"http://example.org/network-graphs/68/relationships/nodes","related":"http://example.org/network-graphs/68/nodes"}}}}}

----------------------

  * type - Should always be set to <code>network-graphs</code>
  * sheet - Sheet id
  * network-template - Network Template id

Request:

  Content-Type: application/vnd.api+json

  {"data":{"type":"network-graphs","relationships":{"sheet":{"data":{"type":"sheets","id":169}},"network-template":{"data":{"type":"network-templates","id":111}}}}}
