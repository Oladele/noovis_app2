Create a network graph
----------------------

Parameters:
  * type - Should always be set to <code>network-graphs</code>
  * sheet - Sheet id
  * network-template - Network Template id

Request:
  POST /network-graphs
  Content-Type: application/vnd.api+json

  {"data":{"type":"network-graphs","relationships":{"sheet":{"data":{"type":"sheets","id":492}},"network-template":{"data":{"type":"network-templates","id":323}}}}}

Response:
  Status: 201 Created
  Content-Type: application/vnd.api+json

  {"data":{"id":"207","type":"network-graphs","links":{"self":"http://example.org/network-graphs/207"},"attributes":{"nodes":[],"edges":[]},"relationships":{"sheet":{"links":{"self":"http://example.org/network-graphs/207/relationships/sheet","related":"http://example.org/network-graphs/207/sheet"}},"network-template":{"links":{"self":"http://example.org/network-graphs/207/relationships/network-template","related":"http://example.org/network-graphs/207/network-template"}},"nodes":{"links":{"self":"http://example.org/network-graphs/207/relationships/nodes","related":"http://example.org/network-graphs/207/nodes"}}}}}
