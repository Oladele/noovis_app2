Create a network graph
----------------------

Parameters:
  * type - Should always be set to <code>network-graphs</code>
  * sheet - Sheet id
  * network-template - Network Template id

Request:
  POST /network-graphs
  Content-Type: application/vnd.api+json

  {"data":{"type":"network-graphs","relationships":{"sheet":{"data":{"type":"sheets","id":24}},"network-template":{"data":{"type":"network-templates","id":16}}}}}

Response:
  Status: 201 Created
  Content-Type: application/vnd.api+json

  {"data":{"id":"10","type":"network-graphs","links":{"self":"http://example.org/network-graphs/10"},"attributes":{"nodes":[],"edges":[]},"relationships":{"sheet":{"links":{"self":"http://example.org/network-graphs/10/relationships/sheet","related":"http://example.org/network-graphs/10/sheet"}},"network-template":{"links":{"self":"http://example.org/network-graphs/10/relationships/network-template","related":"http://example.org/network-graphs/10/network-template"}},"nodes":{"links":{"self":"http://example.org/network-graphs/10/relationships/nodes","related":"http://example.org/network-graphs/10/nodes"}}}}}


Update a network-graph
----------------------

Parameters:
  * type - Should always be set to <code>network-graphs</code>
  * sheet - Sheet id
  * network-template - Network Template id
  * id - The id of the network-graph

Request:
  PATCH /network-graphs/11
  Content-Type: application/vnd.api+json

  {"data":{"type":"network-graphs","relationships":{"sheet":{"data":{"type":"sheets","id":26}},"network-template":{"data":{"type":"network-templates","id":18}}},"id":"11"}}

Response:
  Status: 200 OK
  Content-Type: application/vnd.api+json

  {"data":{"id":"11","type":"network-graphs","links":{"self":"http://example.org/network-graphs/11"},"attributes":{"nodes":[],"edges":[]},"relationships":{"sheet":{"links":{"self":"http://example.org/network-graphs/11/relationships/sheet","related":"http://example.org/network-graphs/11/sheet"}},"network-template":{"links":{"self":"http://example.org/network-graphs/11/relationships/network-template","related":"http://example.org/network-graphs/11/network-template"}},"nodes":{"links":{"self":"http://example.org/network-graphs/11/relationships/nodes","related":"http://example.org/network-graphs/11/nodes"}}}}}


Delete a network-graph
----------------------

Parameters:
  * type - Should always be set to <code>network-graphs</code>
  * sheet - Sheet id
  * network-template - Network Template id
  * id - The id of the network-graph

Request:
  DELETE /network-graphs/12
  Content-Type: application/vnd.api+json

  {"data":{"type":"network-graphs","relationships":{"sheet":{"data":{"type":"sheets","id":28}},"network-template":{"data":{"type":"network-templates","id":20}}},"id":"12"}}

Response:
  Status: 204 No Content
