Create a network graph
----------------------

Parameters:
  * type - Should always be set to <code>network-graphs</code>
  * sheet - Sheet id
  * network-template - Network Template id

Request:
  POST /network-graphs
  Content-Type: application/vnd.api+json

  {"data":{"type":"network-graphs","relationships":{"sheet":{"data":{"type":"sheets","id":1207}},"network-template":{"data":{"type":"network-templates","id":814}}}}}

Response:
  Status: 201 Created
  Content-Type: application/vnd.api+json

  {"data":{"id":"524","type":"network-graphs","links":{"self":"http://example.org/network-graphs/524"},"relationships":{"sheet":{"links":{"self":"http://example.org/network-graphs/524/relationships/sheet","related":"http://example.org/network-graphs/524/sheet"}},"network-template":{"links":{"self":"http://example.org/network-graphs/524/relationships/network-template","related":"http://example.org/network-graphs/524/network-template"}},"nodes":{"links":{"self":"http://example.org/network-graphs/524/relationships/nodes","related":"http://example.org/network-graphs/524/nodes"}}}}}


Delete a network-graph
----------------------

Parameters:
  * type - Should always be set to <code>network-graphs</code>
  * sheet - Sheet id
  * network-template - Network Template id
  * id - The id of the network-graph

Request:
  DELETE /network-graphs/525
  Content-Type: application/vnd.api+json

  {"data":{"type":"network-graphs","relationships":{"sheet":{"data":{"type":"sheets","id":1209}},"network-template":{"data":{"type":"network-templates","id":816}}},"id":"525"}}

Response:
  Status: 204 No Content
