Create a network graph
----------------------

Parameters:
  * type - Should always be set to <code>network-graphs</code>
  * sheet - Sheet id
  * network-template - Network Template id

Request:
  POST /network-graphs
  Content-Type: application/vnd.api+json

  {"data":{"type":"network-graphs","relationships":{"sheet":{"data":{"type":"sheets","id":11}},"network-template":{"data":{"type":"network-templates","id":9}}}}}

Response:
  Status: 201 Created
  Content-Type: application/vnd.api+json

  {"data":{"id":"7","type":"network-graphs","links":{"self":"http://example.org/network-graphs/7"},"attributes":{"nodes":[],"edges":[]},"relationships":{"sheet":{"links":{"self":"http://example.org/network-graphs/7/relationships/sheet","related":"http://example.org/network-graphs/7/sheet"}},"network-template":{"links":{"self":"http://example.org/network-graphs/7/relationships/network-template","related":"http://example.org/network-graphs/7/network-template"}},"nodes":{"links":{"self":"http://example.org/network-graphs/7/relationships/nodes","related":"http://example.org/network-graphs/7/nodes"}}}}}

----------------------

Parameters:
  * type - Should always be set to <code>network-graphs</code>
  * sheet - Sheet id
  * network-template - Network Template id
  * id - The id of the network-graph

Request:
  PATCH /network-graphs/8
  Content-Type: application/vnd.api+json

  {"data":{"type":"network-graphs","relationships":{"sheet":{"data":{"type":"sheets","id":13}},"network-template":{"data":{"type":"network-templates","id":11}}},"id":"8"}}
