Create a network graph
----------------------

Parameters:
  * type - Should always be set to <code>network-graphs</code>
  * sheet - Sheet id
  * network-template - Network Template id

Request:
  POST /network-graphs
  Content-Type: application/vnd.api+json

  {"data":{"type":"network-graphs","relationships":{"sheet":{"data":{"type":"sheets","id":298}},"network-template":{"data":{"type":"network-templates","id":213}}}}}

Response:
  Status: 201 Created
  Content-Type: application/vnd.api+json

  {"data":{"id":"136","type":"network-graphs","links":{"self":"http://example.org/network-graphs/136"},"attributes":{"nodes":[],"edges":[]},"relationships":{"sheet":{"links":{"self":"http://example.org/network-graphs/136/relationships/sheet","related":"http://example.org/network-graphs/136/sheet"}},"network-template":{"links":{"self":"http://example.org/network-graphs/136/relationships/network-template","related":"http://example.org/network-graphs/136/network-template"}},"nodes":{"links":{"self":"http://example.org/network-graphs/136/relationships/nodes","related":"http://example.org/network-graphs/136/nodes"}}}}}


Delete a network-graph
----------------------

Parameters:
  * type - Should always be set to <code>network-graphs</code>
  * sheet - Sheet id
  * network-template - Network Template id
  * id - The id of the network-graph

Request:
  DELETE /network-graphs/137
  Content-Type: application/vnd.api+json

  {"data":{"type":"network-graphs","relationships":{"sheet":{"data":{"type":"sheets","id":300}},"network-template":{"data":{"type":"network-templates","id":215}}},"id":"137"}}

Response:
  Status: 204 No Content
