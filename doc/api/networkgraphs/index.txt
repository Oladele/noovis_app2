Update a network-graph
----------------------

Parameters:
  * type - Should always be set to <code>network-graphs</code>
  * sheet - Sheet id
  * network-template - Network Template id
  * id - The id of the network-graph

Request:
  PATCH /network-graphs/159
  Content-Type: application/vnd.api+json

  {"data":{"type":"network-graphs","relationships":{"sheet":{"data":{"type":"sheets","id":349}},"network-template":{"data":{"type":"network-templates","id":254}}},"id":"159"}}

Response:
  Status: 200 OK
  Content-Type: application/vnd.api+json

  {"data":{"id":"159","type":"network-graphs","links":{"self":"http://example.org/network-graphs/159"},"attributes":{"nodes":[],"edges":[]},"relationships":{"sheet":{"links":{"self":"http://example.org/network-graphs/159/relationships/sheet","related":"http://example.org/network-graphs/159/sheet"}},"network-template":{"links":{"self":"http://example.org/network-graphs/159/relationships/network-template","related":"http://example.org/network-graphs/159/network-template"}},"nodes":{"links":{"self":"http://example.org/network-graphs/159/relationships/nodes","related":"http://example.org/network-graphs/159/nodes"}}}}}
