Update a network-graph
----------------------

Parameters:
  * type - Should always be set to <code>network-graphs</code>
  * sheet - Sheet id
  * network-template - Network Template id
  * id - The id of the network-graph

Request:
  PATCH /network-graphs/489
  Content-Type: application/vnd.api+json

  {"data":{"type":"network-graphs","relationships":{"sheet":{"data":{"type":"sheets","id":1139}},"network-template":{"data":{"type":"network-templates","id":769}}},"id":"489"}}

Response:
  Status: 200 OK
  Content-Type: application/vnd.api+json

  {"data":{"id":"489","type":"network-graphs","links":{"self":"http://example.org/network-graphs/489"},"relationships":{"sheet":{"links":{"self":"http://example.org/network-graphs/489/relationships/sheet","related":"http://example.org/network-graphs/489/sheet"}},"network-template":{"links":{"self":"http://example.org/network-graphs/489/relationships/network-template","related":"http://example.org/network-graphs/489/network-template"}},"nodes":{"links":{"self":"http://example.org/network-graphs/489/relationships/nodes","related":"http://example.org/network-graphs/489/nodes"}}}}}
