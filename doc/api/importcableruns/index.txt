Upload sheet
------------

Parameters:
  * file - Workbook File
  * sheet - Sheet Name
  * building_id - Building id

Request:
  POST /import_cable_run
  Content-Type: multipart/form-data

  {"data":{"file":{"content_type":{"content_type":"application/vnd.ms-excel","binary":true},"original_filename":"test.xls","tempfile":["This is a test file\n"]},"sheet":"Sheet1","building_id":1311}}

Response:
  Status: 201 Created
  Content-Type: application/json; charset=utf-8

  {
    "message": "Successfully created cable runs"
  }
