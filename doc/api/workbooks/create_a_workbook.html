Workbooks API

Create a workbook

POST /workbooks

Parameters

Name Description
relationships[network-site] Network site id
type Should always be set to <code>workbooks</code>
attributes[name] Workbook Name

Request

Headers

Content-Type: application/vnd.api+json

Route

POST /workbooks

Body

{"data":{"relationships":{"network-site":{"data":{"type":"network-sites","id":35}}},"type":"workbooks","attributes":{"name":"Workbook Name"}}}

Response

Headers

Content-Type: application/vnd.api+json

Status

201 Created

Body

{"data":{"id":"16","type":"workbooks","links":{"self":"http://example.org/workbooks/16"},"attributes":{"name":"Workbook Name"},"relationships":{"network-site":{"links":{"self":"http://example.org/workbooks/16/relationships/network-site","related":"http://example.org/workbooks/16/network-site"}},"sheets":{"links":{"self":"http://example.org/workbooks/16/relationships/sheets","related":"http://example.org/workbooks/16/sheets"}}}}}
