Buildings API

Update a building

PATCH /buildings/:building_id

Parameters

Name Description
relationships[network-site] Network site id
type Should always be set to <code>buildings</code>
attributes[name] Building Name
attributes[lat] Building Latitude
attributes[lng] Building Longitude
id The id of the building

Request

Headers

Content-Type: application/vnd.api+json

Route

PATCH /buildings/17

Body

{"data":{"relationships":{"network-site":{"data":{"type":"network-sites","id":44}}},"type":"buildings","attributes":{"name":"Updated Building Name"},"id":"17"}}

Response

Headers

Content-Type: application/vnd.api+json

Status

200 OK

Body

{"data":{"id":"17","type":"buildings","links":{"self":"http://example.org/buildings/17"},"attributes":{"name":"Updated Building Name","lat":"39.290139","lng":"-76.614766","description":"MyText"},"relationships":{"network-site":{"links":{"self":"http://example.org/buildings/17/relationships/network-site","related":"http://example.org/buildings/17/network-site"}},"sheets":{"links":{"self":"http://example.org/buildings/17/relationships/sheets","related":"http://example.org/buildings/17/sheets"}}}}}
